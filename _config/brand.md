# Brand Identity — FootballHeritage

## Project Identity
FootballHeritage is a sports betting intelligence platform combining real-time odds, AI-powered predictions, and responsible gambling tools. The brand bridges data-driven analytics with the passion of sports fandom.

## Tone
- **Confident** — backed by data and models
- **Transparent** — clear about prediction methodology and limitations
- **Responsible** — gambling awareness is core to the experience
- **Modern** — clean, fast, data-forward UI

## Visual Direction
- **Colors:** Deep navy / emerald green / gold accent / white
- **Typography:** Clean sans-serif for UI, monospace for data displays
- **Imagery:** Sports-focused data visualizations, clean dashboards, minimal decorative elements
- **UI Vibe:** Dark mode dashboard aesthetic with vibrant data highlights

## Target Audience
- Sports bettors seeking data-driven insights
- Analytics enthusiasts interested in prediction models
- Responsible gambling advocates
- Sports data analysts and researchers

## Key Messaging
- "Bet smarter with AI-powered predictions"
- "Real-time odds. Data-driven insights. Responsible play."
- "Your edge in sports intelligence"
