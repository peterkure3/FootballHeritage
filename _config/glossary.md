# Glossary — FootballHeritage

| Term | Definition |
|------|------------|
| **Odds** | Numerical representation of event probability and potential payout |
| **Spread** | Point spread — handicap applied to equalize betting on uneven matchups |
| **Over/Under (Total)** | Bet on whether combined score exceeds or falls below a line |
| **Parlay** | Single bet linking multiple outcomes; all must win for payout |
| **Moneyline** | Straight bet on which team/player wins the event |
| **Push** | Tie result where all stakes are refunded |
| **Vig (Vigorish)** | House commission built into odds |
| **Responsible Gambling** | Features promoting safe play: deposit limits, self-exclusion, reality checks |
| **ETL** | Extract, Transform, Load — data pipeline process |
| **Model Accuracy** | Percentage of correct predictions vs actual outcomes |
| **ELO Rating** | Rating system for relative skill levels of teams/players |
| **Holdout Data** | Data set withheld from training used to evaluate model performance |
| **Genkit Flow** | Genkit-defined chain of AI operations with prompt templates and tool calls |
| **Scraper** | Automated data collection from web sources without official API |
| **CORS** | Cross-Origin Resource Sharing — browser security policy for API access |
| **JWT** | JSON Web Token — stateless authentication mechanism |
